## À propos

Site portfolio one-page entièrement fait en HTML/CSS vanilla — sans framework, sans dépendance. Il présente mon parcours, mes compétences et mes projets, avec un mode clair et un mode sombre.

## Fonctionnalités

- 🌗 Thème clair / sombre avec mémorisation du choix (localStorage)
- 📄 Téléchargement du CV en un clic
- 🔗 Liens directs vers les projets GitHub
- 📱 Responsive mobile
- ⚡ Zéro dépendance — HTML & CSS pur

## Stack

| Technologie | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 (variables, grid, flexbox) | Style & thèmes |
| JavaScript vanilla | Toggle thème, localStorage |
| Google Fonts — Inter | Typographie |

## Lancer en local

Aucune installation requise. Clonez et ouvrez directement dans votre navigateur :

```bash
git clone https://github.com/Fatima-Zahra7/fzbekraoui-portfolio.git
cd fzbekraoui-portfolio
open portfolio_fatima_zahra.html   # macOS
# ou double-cliquez sur portfolio_fatima_zahra.html sous Windows
```

Avec VS Code, installez l'extension **Live Server** et faites clic droit → *Open with Live Server* pour un rechargement automatique.

## Déploiement

Le site est hébergé via **GitHub Pages** avec un domaine personnalisé configuré via OVHcloud.

Pour redéployer : tout commit sur la branche `main` est automatiquement mis en ligne.
